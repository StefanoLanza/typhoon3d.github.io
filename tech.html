TECHNOLOGY

Typhoon is written in C/C++, Lua and HLSL. It has been designed with these principles in mind:

- Simplicity
- Robustness
- Efficiency
- Innovation

The engine has a modular architecture which promotes simplicity, elegance, robustness and extensibility. Scenes are organized as hierarchies of game objects, similarly to Unity. Our component-based architecture supports logical hierarchies, prefabricated entities and components, inheritance. Entities and components can be serialized, exposed to Lua and edited at run time (using our code reflection library). All components are programmed in C++ for efficiency and then exposed to Lua. Scripts are written exclusively in Lua and define the application logic, which remains completely separated from the engine code.
