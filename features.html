Component Based

Scenes are organized as hierarchies of game objects, which are containers of components and scripts, similarly to Unity. This architecture supports logical hierarchies, prefabricated entities and inheritance. Entities and components can be serialized, exposed to Lua and edited at run time (thanks to our code reflection library). Most components are programmed in C and structured in memory for maximum efficiency.
